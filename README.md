# BakeOS POS System With PHP for Bakery Management

## Overview

BakeOS is a modern Bakery Management and Point of Sale (POS) System built using PHP, MySQL, HTML, CSS, JavaScript, and Tailwind CSS.

This project is designed for:

* Bakery businesses
* Small food shops
* School thesis/capstone projects
* Learning PHP POS development
* Real-world bakery inventory and sales management

BakeOS includes:

* POS Checkout System
* Inventory Management
* Stock IN / OUT Monitoring
* Damage Monitoring
* Sales Reports
* User Management
* Dashboard Analytics
* Mobile-Friendly POS Interface

# Installation Guide

## 1. Clone or Download Project

Place the project inside your web server directory.

Example for XAMPP:

\htdocs\bakeos

---

## 2. Create Database

Open phpMyAdmin and create a database:

bakeos

---

## 3. Import SQL Database

Import the provided `.sql` database file into phpMyAdmin.

---

## 4. Configure Database Connection

Open:

db.php

Update database credentials if necessary:

```php
$host = "localhost";
$dbname = "bakeos";
$username = "root";
$password = "";
```

---

## 5. Run The Project

Open browser:

http://localhost/bakeos
